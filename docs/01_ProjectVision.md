# LegacyLens AI

## Project Vision

LegacyLens AI is an AI-powered web platform designed to help software engineers and companies understand, analyze, and modernize legacy software systems.

The platform reduces the time required to understand large codebases by automatically generating documentation, identifying technical debt, detecting deprecated technologies, and providing AI-powered modernization recommendations.

Instead of spending days exploring an unfamiliar project, developers can upload a project or connect a Git repository and receive a structured analysis within minutes.

---

# Problem Statement

Many organizations still rely on legacy software that has been maintained for years without proper documentation.

These systems often suffer from:

- Missing or outdated documentation
- Large and complex codebases
- Deprecated frameworks and libraries
- High technical debt
- Difficult onboarding for new developers
- Slow modernization processes

Understanding such systems usually requires many hours or even weeks.

---

# Proposed Solution

LegacyLens AI provides an intelligent assistant that analyzes software projects and generates useful insights for developers.

The platform combines static code analysis with Large Language Models (LLMs) to produce understandable and actionable reports.

The first version of the platform will focus on helping developers understand existing projects before attempting modernization.

# Core Features (MVP)

## Project Management

- Create projects
- Upload ZIP source code
- Connect Git repositories (future)

## Project Analysis

- Detect programming language
- Detect framework
- Analyze project structure
- Display folder hierarchy

## AI Documentation

- File summaries
- Module summaries
- Project overview
- README generation

## Code Quality

- Detect code smells
- Detect deprecated APIs
- Identify technical debt
- Suggest improvements

## Modernization

- Generate migration roadmap
- Prioritize modernization tasks
- Estimate modernization complexity

## AI Assistant

- Answer questions about the uploaded project
- Explain files and functions
- Help developers navigate the codebase
